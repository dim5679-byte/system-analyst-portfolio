# Functional Requirements

## Назначение документа

Документ содержит функциональные требования первой версии проекта Smart Home Management Platform.

Требования используются командой разработки при реализации системы и являются основой для подготовки пользовательских историй, API и тестовых сценариев.

---

## Область действия

Документ описывает функциональность MVP.

Требования сгруппированы по функциональным областям системы.

---

## Правила оформления требований

Каждое требование:

- имеет уникальный идентификатор;
- описывает одно действие системы;
- сформулировано однозначно;
- может быть проверено в процессе тестирования.

---

# FR-001 Управление учетной записью

### Требования

**FR-001**

Система должна предоставлять возможность регистрации нового пользователя по адресу электронной почты.

**FR-002**

Система должна предоставлять возможность входа в систему по электронной почте и паролю.

**FR-003**

Система должна предоставлять возможность восстановления пароля.

**FR-004**

Пользователь должен иметь возможность изменить свое имя.

---

# FR-002 Управление домами

### Требования

**FR-005**

Пользователь должен иметь возможность создать новый дом.

**FR-006**

Пользователь должен иметь возможность изменить название дома.

**FR-007**

Пользователь должен иметь возможность удалить дом.

---

# FR-003 Управление комнатами

### Требования

**FR-008**

Пользователь должен иметь возможность создать комнату.

**FR-009**

Пользователь должен иметь возможность изменить название комнаты.

**FR-010**

Пользователь должен иметь возможность удалить комнату.

---

# FR-004 Управление устройствами

### Требования

**FR-011**

Пользователь должен иметь возможность подключить устройство посредством QR-кода.

**FR-012**

Пользователь должен иметь возможность удалить устройство.

**FR-013**

Система должна отображать текущее состояние устройства.

---

# FR-005 Управление состоянием устройств

### Требования

**FR-014**

Пользователь должен иметь возможность включить устройство.

**FR-015**

Пользователь должен иметь возможность выключить устройство.

**FR-016**

Пользователь должен иметь возможность управлять устройствами внутри выбранной комнаты.

---
