# Scope

## Назначение документа

Документ определяет границы первой версии проекта Smart Home Management Platform и фиксирует функциональность, которая входит в MVP.

---

## Границы проекта

В рамках первой версии разрабатывается мобильное приложение для управления устройствами умного дома.

Проект охватывает регистрацию пользователей, создание домов и комнат, подключение устройств и выполнение базовых операций управления.

Функциональность, не относящаяся к основным пользовательским сценариям, переносится на последующие версии продукта.

---

## Что входит в MVP

В первую версию входят следующие возможности:

- регистрация и авторизация пользователей;
- восстановление пароля;
- управление профилем пользователя;
- создание нескольких домов;
- создание комнат;
- редактирование комнат;
- подключение устройств по QR-коду;
- удаление устройств;
- управление состоянием устройств;
- объединение устройств по комнатам.

---

## Что не входит в MVP

На первый релиз не планируется:

- автоматизация сценариев;
- голосовое управление;
- поддержка датчиков;
- интеграция с системами безопасности;
- уведомления;
- управление через веб-интерфейс;
- совместная работа нескольких пользователей;
- аналитика энергопотребления.

---

## Ограничения проекта

Для первой версии определены следующие ограничения:

- поддерживается только мобильное приложение;
- поддерживаются базовые типы устройств;
- один пользователь является владельцем дома;
- работа приложения требует подключения к интернету.

---

## Предположения

При разработке предполагается, что:

- устройства поддерживают стандартный механизм подключения;
- пользователь обладает стабильным интернет-соединением;
- производители предоставляют API для управления устройствами.

---

## Зависимости

Проект зависит от:

- доступности API производителей устройств;
- мобильных платформ Android и iOS;
- облачной инфраструктуры для хранения данных.

---

## Критерии завершения MVP

Первая версия считается завершенной, если пользователь может:

- зарегистрироваться;
- создать дом;
- добавить комнаты;
- подключить устройство;
- управлять устройствами;
- удалить устройство.

---
