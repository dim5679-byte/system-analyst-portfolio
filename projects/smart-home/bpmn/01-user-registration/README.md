# User Registration

## Назначение

Диаграмма описывает процесс регистрации нового пользователя в системе Smart Home Management Platform.

---

## Цель процесса

Предоставить пользователю возможность создать учетную запись для дальнейшего управления устройствами умного дома.

---

## Основные участники

- Пользователь
- Мобильное приложение
- Backend API

---

## Входные данные

- Email
- Пароль

---

## Результат

Созданная учетная запись пользователя.

---

## Статус

В работе
