## О разделе

В данном разделе собрана вся документация по анализу требований проекта **Smart Home Platform**.

Документы расположены в последовательности, соответствующей жизненному циклу анализа требований — от формирования видения продукта до описания бизнес-правил.

## Правила оформления

Каждый документ оформляется как самостоятельный аналитический артефакт и содержит:

- назначение документа;
- версию;
- статус;
- описание;
- содержание;
- ссылки на связанные документы.
