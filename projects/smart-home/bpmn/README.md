# BPMN

## Назначение раздела

В разделе представлены BPMN-диаграммы основных бизнес-процессов Smart Home Management Platform.

Диаграммы используются для детализации пользовательских сценариев, согласования логики работы системы и передачи требований разработчикам.

## План развития

Планируется моделирование процессов:

- регистрация пользователя;
- авторизация;
- создание дома;
- создание комнаты;
- подключение устройства;
- управление устройством;
- удаление устройства.
