# Smart Home

## О проекте

**Smart Home** — демонстрационный проект информационной системы управления умным домом.

Проект разработан как пример полного цикла работы системного аналитика — от сбора требований до проектирования архитектуры, моделей данных и API.

---

## Цель проекта

Создать систему управления устройствами умного дома с возможностью:

- регистрации пользователей;
- управления домами и комнатами;
- подключения устройств;
- совместного доступа;
- управления освещением;
- построения масштабируемой архитектуры.
