# Smart Home

## О проекте

**Smart Home** — портфельный проект системы управления умным домом.

Проект демонстрирует полный цикл работы системного аналитика: от анализа требований до проектирования бизнес-процессов, модели данных, API и архитектурных артефактов.

---

## Цель проекта

Создать систему управления устройствами умного дома с возможностью:

- регистрации пользователей;
- управления домами и комнатами;
- подключения устройств;
- совместного доступа;
- управления освещением;
- построения масштабируемой архитектуры.
