# Vision

This document describes the purpose, target audience, and value of the System Analyst Portfolio.

## Purpose

Create a professional portfolio that demonstrates system analysis skills through structured projects, artifacts, and documentation.

## Target Audience

- HR specialists
- IT recruiters
- Team Leads
- Senior System Analysts
- Product and Project Managers

## Value

The portfolio should help an employer quickly understand:

- who I am;
- what skills I have;
- what projects I worked on;
- what artifacts I can prepare;
- how I structure information.
