# Design Principles

## 1. Clear Navigation

Any important information should be reachable within three clicks.

## 2. Project-Based Structure

The portfolio is organized around projects first, and artifact types second.

## 3. Consistent Templates

Each project should follow the same structure:

- Overview
- Business Context
- My Role
- Tasks
- Artifacts
- Result

## 4. No Random Files

Every file should have a clear purpose.

## 5. Employer-Oriented Presentation

The portfolio should be easy to review for HR specialists, recruiters, and technical leads.
